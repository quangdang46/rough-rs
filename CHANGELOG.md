# Changelog

## 0.1.0 - Unreleased

- Port rough.js-style primitives: line, rectangle, ellipse, circle, linear path,
  polygon, arc, curve, and SVG path data.
- Add hachure, solid, zigzag, cross-hatch, dots, dashed, and zigzag-line fills.
- Add rough.js-compatible seeded renderer output, including rough.js-compatible
  nondeterministic dot-fill center placement.
- Add rough.js reference fixture generation, expanded supported-surface parity
  coverage, and strict numeric parity tests for deterministic cases.
- Add examples and Criterion benchmarks.
